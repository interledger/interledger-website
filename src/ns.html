#hasPaymentPointer

Predicate to link from a WebID to a node of type InterledgerPaymentPointer.

#paymentPointerValue

Predicate to link from a node of type InterledgerPaymentPointer to a payment pointer string value.

#InterledgerPaymentPointer

RDF class to describe Interledger Pointer Pointers in RDF.

Example


@prefix ilp: <https://interledger.org/ns#>.
<#me> ilp:hasPaymentPointer [
    a ilp:InterledgerPaymentPointer;
    ilp:paymentPointerValue "$example.com".
  ].
